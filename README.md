# Spring-crud-implementation
Basic CRUD application
Has Docker implementation with Spring and Mysql


Docker Commands:
docker build -t ankit26031993/dockerdemo:0.0.1 .
docker run -p 8080:8080 --name dockerdemo ankit26031993/dockerdemo:0.0.1
docker-compose up
docker exec <container id> #to enter mysql env and execute queries
